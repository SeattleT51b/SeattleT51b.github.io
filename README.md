# SeattleT51b 

## Features

- **Hero Section with Image**: Full-width hero image with overlay title
- **About Section**: Brief project description
- **Photo Gallery Slideshow**: Auto-rotating slideshow with manual controls and navigation dots
- **FAQ Accordion**: Collapsible questions with expandable answers
- **Contributors Section**: Grid display of major contributors with photos
- **Parts & Shopping List**: Table with parts, quantities, suppliers, and purchase links
- **BlueSky Social Integration**: Link to BlueSky account in footer
- **Responsive Design**: Mobile-first approach, works on all screen sizes
- **Smooth Navigation**: Smooth scrolling between sections

## Tech Stack

- HTML5
- CSS3 (with animations and responsive design)
- Vanilla JavaScript (no dependencies)
